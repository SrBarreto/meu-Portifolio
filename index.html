Sobre Mim

Olá! Meu nome é Roger Barreto, Desenvolvedor Front-end em transição de carreira com forte raciocínio lógico e disciplina, herdados da experiência anterior. Estou ativamente familiarizado com Python e aplico consistentemente Git para versionamento de código, além de compreender o fluxo de trabalho Ágil e a importância do Code Review. Busco uma posição Júnior onde possa aplicar minha dedicação e capacidade analítica, contribuindo com a qualidade de código e aprendendo sob a mentoria de um time de desenvolvimento.
